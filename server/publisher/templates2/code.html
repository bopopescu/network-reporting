Code Integration

  1. You will need to install the MoPub SDK first. If you have not done so, see the MoPub Getting Started instructions on Google Code.

  2. Add the following XML block to your layout definition (eg. main.xml): 

                <com.mopub.mobileads.MoPubView
              android:id="@+id/adview"
              android:layout_width="fill_parent"
              android:layout_height="50px"/>

  3. Add the following code snippet in your Activity's onCreate() instance (or another suitable place): 

                  MoPubView mpv = (MoPubView) findViewById(R.id.adview);
              mpv.setAdUnitId("agltb3B1Yi1pbmNyDAsSBFNpdGUY2aQGDA");
              mpv.loadAd();

  4. For more options, or if you have trouble, see the MoPub Google Code project.
